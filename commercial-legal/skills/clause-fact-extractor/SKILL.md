# clause-fact-extractor

## Purpose
Pulls structured data from a contract document for review.

## Inputs
- Raw contract text

## Steps
1. Identify and extract all key clauses
2. Map each clause to the relevant playbook provision
3. Return structured list of clause findings

## Outputs
- Structured list of extracted clauses with playbook mapping

## Notes for Lawyers
This skill extracts — it does not evaluate. Evaluation is done by the review agents.
